# PRESTO

Unified immunoinformatics for MHC presentation and TCR recognition.

## Quickstart

Run the CLI from the repo root:

```bash
python -m presto --help
python -m presto data list
```

## What’s included

- Data download + processing utilities
- Multi-task model architecture for pMHC presentation and TCR recognition
- Training demos (synthetic, curriculum)
- Predictor API for inference

See the CLI guide for more examples.
